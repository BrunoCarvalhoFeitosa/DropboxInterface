## Sobre o projeto

![DropBoxClone](https://user-images.githubusercontent.com/46093815/134263444-1d45516e-28d6-4147-923e-e0f387f46641.png)

Este app é um clone da antiga interface da homepage do Dropbox, possui algumas animações e informações à respeito dos serviços oferecidos pela plataforma.
### Feito com

* [React.js](https://pt-br.reactjs.org/)
* [Styled-components](https://styled-components.com/)

### Hospedagem

O app ainda não foi hospedado em nenhuma plataforma, porém você poderá cloná-lo. Basta acompoanhar o próximo tópico.


<!-- GETTING STARTED -->
## Iniciando o projeto

Primeiramente será necessário clonar este projeto em (https://github.com/BrunoCarvalhoFeitosa/Dropbox-Interface.git), após o download será necessário abrir este projeto no seu
editor e no terminal digitar npm install ou yarn install, posteriormente é só executar o script yarn start ou npm run start. 

### Pré-requisitos

* npm
  ```sh
  npm install npm@latest -g
  ```

### Instalação

1. Clone o repositório
   ```sh
   git clone https://github.com/BrunoCarvalhoFeitosa/Dropbox-Interface.git
   ```
2. Instale os pacotes do NPM
   ```sh
   npm install ou yarn install
   ```
   
3. Inicie o projeto
   ```sh
   npm run start ou yarn start
   ```
